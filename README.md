## 🪜 How to Publish the Site (Takes 5 Minutes)

### Step 1 — Create the Repository
1. Go to [github.com/new](https://github.com/new)  
2. **Repository name:** `curbsideautocare`  
3. Keep it **Public**  
4. Click **Create repository**

### Step 2 — Upload Your Files
1. Click **Add file → Upload files**  
2. Upload these two files:
   - `index.html`  ← (the site file from ChatGPT)
   - `README.md`  ← (this file)
3. Click **Commit changes**

### Step 3 — Enable GitHub Pages
1. In the repo, click **Settings → Pages**  
2. Under **Source**, choose “Deploy from a branch”  
3. Under **Branch**, select `main` → `/ (root)`  
4. Click **Save**

### Step 4 — Wait 1–2 Minutes
GitHub will automatically publish your site.
